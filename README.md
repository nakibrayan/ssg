### dependencies

- none

---

### build

```bash
git clone https://github.com/nakibrayan/ssg.git
```

```bash
cd ssg
```

```bash
cc ssg.c -o ssg
```

---

### install

go through out the `build` tutorial and then run this commands

```bash
sudo mkdir -p /usr/local/bin
```

```bash
sudo mv ssg /usr/local/bin
```
